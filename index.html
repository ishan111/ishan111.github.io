Website + CleverTap Web Push Demo

This demo shows the full journey: identify a visitor, track web events, request browser push permission, and then target that user from CleverTap.

Demo outcome Known user → tracked event → subscribed browser → push delivered
  1. Open the site on Chrome.
  2. Identify a demo user.
  3. Trigger 1–2 events.
  4. Click Enable Web Push.
  5. Send a push from CleverTap.
